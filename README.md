# E-LUDIK

Соцсеть ставок с публичными профилями:

- регистрация и вход
- личный кабинет со статистикой
- история ставок
- друзья и сравнение результатов
- публичный профиль `/u/<username>`
- раздел правил ставок

## Запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

По умолчанию приложение запускается на `http://localhost:8080`.

## Переменные окружения

- `HOST` (по умолчанию `127.0.0.1`)
- `PORT` (по умолчанию `8080`)
- `DEBUG` (`1` или `0`)
- `SESSION_SECRET` (рекомендуется задать в проде)

Для локальной разработки открывай именно `http://127.0.0.1:8080` или `http://localhost:8080`.
