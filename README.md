# E-LUDIK

## 1) Описание приложения

E-LUDIK — это социальная платформа для ставок, ориентированная на личную статистику и обмен результатами с друзьями. Пользователи сохраняют свои ставки, анализируют результаты, сравнивают достижения с другими и ведут публичный профиль.

Приложение построено на Python + Bottle и использует SQLite для хранения данных.

## 2) Установка и требования

### Системные требования

- Python 3.9 или выше
- Виртуальное окружение (рекомендуется)
- SQLite (входит в стандартную библиотеку Python)

## 3) Обзор интерфейса

### Главное меню

- `Правила` — страница с рекомендациями по дисциплине ставок.
- `Кабинет` — личный кабинет с основной статистикой и формой для ставок.
- `Сравнение` — рейтинг и результаты друзей.
- `Публичный профиль` — общий профиль пользователя.
- `Войти` / `Регистрация` — авторизация и создание аккаунта.

### Личный кабинет

В личном кабинете доступны:

- статистика по ставкам: общее количество, винрейт, чистая прибыль, ROI;
- форма для добавления новой ставки;
- таблица последних ставок с результатами;
- блок друзей с возможностью добавления и удаления;
- активность друзей;
- редактирование профиля: аватар и биография.

### Публичный профиль

- открытая страница пользователя по адресу `/u/<username>`;
- отображает данные профиля, биографию и последние ставки;
- позволяет другим пользователям добавить владельца в друзья.

### Правила ставок

- содержит базовые рекомендации по банкроллу, фиксированию ставок, формулы и ограничения;
- помогает вести ставки более осознанно и дисциплинированно.

## 4) Основные функции

### Регистрация и вход

- регистрация нового пользователя с проверкой имени и пароля;
- безопасная авторизация через имя пользователя и пароль;
- сессии хранятся в защищенной cookie.

### Управление ставками

- добавление ставки с названием, дисциплиной, коэффициентом, суммой, результатом и комментарием;
- подсчет прибыли автоматически;
- хранение истории ставок в базе данных.

### Статистика и анализ

- расчет винрейта, ROI, общей суммы ставок и чистой прибыли;
- отображение последних ставок и результатов;
- возможности сравнения с друзьями.

### Друзья и социальный обмен

- добавление пользователей в друзья по нику;
- просмотр активности друзей;
- сравнение результатов с друзьями в рейтинге.

### Редактирование профиля

- заполнение и обновление био;
- добавление ссылки на аватар;
- удобный переход к публичному профилю.

## 5) Запуск и конфигурация

### Переменные окружения

- `HOST` — адрес хоста (по умолчанию `127.0.0.1`).
- `PORT` — порт сервера (по умолчанию `8080`).
- `DEBUG` — режим отладки (`1` или `0`).
- `SESSION_SECRET` — секрет для подписи сессионной cookie.

### Пример запуска с настройками

```powershell
$env:HOST="0.0.0.0"
$env:PORT="8080"
$env:DEBUG="1"
$env:SESSION_SECRET="your-secret-value"
python app.py
```

## 6) Структура проекта

- `app.py` — основной сервер и логика приложения;
- `requirements.txt` — зависимости Python;
- `socialbets.db` — локальная база данных SQLite;
- `views/` — HTML-шаблоны страниц;
- `static/` — CSS, JavaScript и медиа-файлы.

## 7) Дополнительно

### Правила ввода данных

- Никнейм должен содержать 3–24 символа, только латиницу, цифры и `_`.
- Пароль должен быть не короче 6 символов.
- Коэффициент ставки должен быть не меньше `1.01`.
- Сумма ставки должна быть не меньше `0.1`.

### Безопасность

- пароли хранятся в виде PBKDF2-HMAC-SHA256 хеша;
- куки сессий подписываются секретным ключом `SESSION_SECRET`;
- база данных SQLite хранится локально в проекте.
