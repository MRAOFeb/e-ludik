Пакет views

Package-level declarations

Шаблоны views не содержат Python-классов, но их можно рассматривать как пакет представлений.

Типы

  • base.tpl — базовый шаблон.
  • dashboard.tpl — личный кабинет.
  • login.tpl — страница входа.
  • register.tpl — регистрация.
  • rules.tpl — правила ставок.
  • public_profile.tpl — публичный профиль.
  • compare.tpl — сравнение.
  • not_found.tpl — страница "Не найдено".

Описание шаблонов

base.tpl

  • содержит шапку сайта и навигацию.
  • подключает styles.css и app.js.
  • отображает уведомления notice.

dashboard.tpl

  • показывает статистику пользователя.
  • содержит форму для добавления ставки.
  • отображает друзей, активность и лидерборд.
  • позволяет редактировать профиль.

public_profile.tpl

  • отображает данные другого пользователя.
  • показывает его биографию, статистику и историю ставок.
  • позволяет добавить пользователя в друзья.

rules.tpl

  • содержит раздел с рекомендациями по банкроллу и дисциплине.
  • описывает основные правила ставок.

Связь с app.py

Каждый шаблон рендерится через вызов template(...) и получает параметры:

  • current_user
  • notice
  • stats, my_bets, friends
  • leaderboard, friends_activity
  • profile_user, profile_stats, profile_bets

Примечание

В дальнейшем можно разделить представления на компоненты и оформить пакет views как настоящий Python-пакет с модульной структурой.
