# E-LUDIK Developer Docs

Проект **E-LUDIK** — это Python/Bottle веб-приложение для социальной сети ставок. Документация оформлена как пакетный API-обзор с акцентом на основные модули и функции.

## Пакеты и модули

- `app` — основной модуль приложения, содержит маршруты, бизнес-логику и взаимодействие с базой данных.
- `views` — набор шаблонов для отображения страниц.
- `static` — статические ресурсы: CSS, JavaScript, изображения.

## Package-level declarations

### Типы

- `app` — основной модуль, содержащий функции и маршруты.
- `views` — пакет шаблонов, которые рендерятся через Bottle.

## Структура

1. `app.py` — весь серверный код приложения.
2. `views/` — HTML-шаблоны для страниц.
3. `static/` — клиентские ресурсы.
4. `socialbets.db` — локальная SQLite база данных.
